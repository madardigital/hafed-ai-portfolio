# حافظ الشرعبي — AI Content Portfolio

معرض أعمال عربي متجاوب يعرض دراسات حالة في صناعة المحتوى بالذكاء الاصطناعي، السرد البصري، الحملات، الفيديو، الموشن جرافيك، والمنصات الرقمية.

## محتوى الموقع

- نبذة مهنية وهوية واضحة.
- طريقة العمل من الموجز إلى الإخراج النهائي.
- ست دراسات حالة تفصيلية.
- روابط أنيقة للفيديوهات والتقارير والمنصات.
- مشاريع رقمية إضافية.
- وضع داكن وفاتح وتصميم متجاوب.

## الرابط المتوقع

بعد تفعيل GitHub Pages:

`https://madardigital.github.io/hafed-ai-portfolio/`

## تفعيل GitHub Pages

من المستودع افتح:

`Settings → Pages → Build and deployment → Source: GitHub Actions`

بعدها سيعمل مسار النشر الموجود داخل `.github/workflows/deploy-pages.yml` تلقائيًا عند كل تحديث على فرع `main`.
