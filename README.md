# حافظ الشرفي — AI Visual Content Portfolio

معرض أعمال عربي متجاوب يوضح خبرتي في صناعة المحتوى البصري باستخدام أدوات الذكاء الاصطناعي، من تطوير الفكرة وكتابة المحتوى والـPrompts إلى توليد الصور والمشاهد والتصميم والموشن والمونتاج.

## محتوى الموقع

- نبذة مهنية واضحة وموجهة إلى مجال صناعة المحتوى البصري بالذكاء الاصطناعي.
- عرض منظم للخدمات ومراحل العمل.
- سبعة نماذج في الفيديو والحملات والموشن والتصميم والمحتوى البصري.
- روابط مباشرة للفيديوهات والتصاميم والتقارير.
- تصميم داكن وعاجي متجاوب مع الهاتف وسطح المكتب.
